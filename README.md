# 💸 SpendBuddy

A modern Android-based Student Finance Manager built using **Java**, **Firebase Authentication**, **Cloud Firestore**, and **Material Design**.

## ✨ Features

- 🔐 Firebase Authentication
- 💰 Budget Management
- 📝 Expense Tracking
- 📊 Analytics Dashboard
- 🎯 Saving Goals
- 🌳 Money Tree Gamification
- 🍃 Leaves Reward System
- 🔥 Daily Challenges
- 📈 Saving Progress Charts
- 👤 Multi-user Support
- ⚙️ Settings Screen

## 🛠 Tech Stack

- Java
- XML
- Android Studio
- Firebase Authentication
- Cloud Firestore
- MPAndroidChart
- Material Design
